# Desafio6_ObtencionDatosNumpy
Una estación meteorológica registra datos de temperatura y humedad cada 5 minutos y los almacena en archivos CSV. Como Data Science debes procesar los datos de un día para realizar un análisis preliminar. Deberás leer los datos, convertirlos a una estructura de datos eﬁciente para cálculo numérico y realizar algunas operaciones básicas.
